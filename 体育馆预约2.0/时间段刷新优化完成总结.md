## 🎯 智能刷新策略

### 策略类型
1. **🥇 cache-first**: 缓存优先，适合一般浏览场景
2. **🚀 api-first**: API优先，适合需要最新数据的场景
3. **🔀 hybrid**: 混合策略，并行请求缓存和API
4. **⚡ force-api**: 强制API刷新，适合支付后等关键场景

### 自动策略选择逻辑
```javascript
// 根据不同场景自动选择最优策略
switch (scenario) {
  case 'booking': return 'api-first'    // 预约时需要最新状态
  case 'payment': return 'force-api'    // 支付后强制刷新
  case 'navigation': return 'cache-first' // 导航时优先缓存
  default: return 'hybrid'              // 默认混合策略
}
```
